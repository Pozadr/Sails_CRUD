# Sail service CRUD

## General
CRUD service with "Sail" model. Used MVC software design pattern.
## Technologies
- Java 11
- Maven
- Spring Boot 2.4.0
- Thymeleaf
- Bootstrap(CSS)
- JQuery/JavaScript
- H2
- MVC
